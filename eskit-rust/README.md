# eskit-rust

Rust 实现，零依赖静态二进制。

## 安装

```bash
cargo install eskit-rust
```

或者从 [Release](https://github.com/deliuou/eskit/releases) 下载静态二进制。

## 指定 es.exe 路径

```bash
export ESKIT_ES_PATH=/mnt/c/你的路径/es.exe
```

## 开发

```bash
git clone https://github.com/deliuou/eskit.git
cd eskit/eskit-rust
cargo build --release
cargo test
```
