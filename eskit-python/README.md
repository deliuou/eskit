# eskit-python

Python 实现，运行在 Python 3.9+ 环境。

## 安装

```bash
pip install eskit-python
```

## 指定 es.exe 路径

```bash
export ESKIT_ES_PATH=/mnt/c/你的路径/es.exe
```

## 开发

```bash
git clone https://github.com/deliuou/eskit.git
cd eskit/eskit-python
pip install -e .
eskit doctor
```
